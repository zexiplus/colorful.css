# colorful.css
colorful.css

[demo page](https://zexiplus.github.io/colorful.css)



### command

* list all

  ```shell
  colorful list 
  ```

* List by like 

  ```shell
  colorful list --by-like 
  ```

* list by view

  ```shell
  colorful list --by-view
  ```

* list by rating

  ```shell
  colorful list --by-rating
  ```

* limit count

  ```shell
  colorful list top 20
  colorful list --by-view top 20
  ```
